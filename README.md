## iRingg 是什么？

iRingg 是 Softorino 开发的一款 Windows 铃声制作工具，核心功能就两件事：**剪铃声** + **传到手机**。

打开软件，把音频文件拖进去，拖动波形选区间，加个淡入淡出，点一下推送——完成。整个流程基本不超过两分钟。

它支持的音频格式相当全，MP3、M4A、WAV、FLAC 都没问题。如果手头没有合适的音频，软件内置了 YouTube 和 SoundCloud 搜索，找到喜欢的歌，直接在软件里截取片段，不用另外下载。

还有个叫 **6-Sense** 的功能，会根据你的 iTunes 音乐库和 iPhone 曲库分析你的口味，主动给你推荐适合做铃声的曲目，有点像音乐推荐算法跑到铃声软件里了。

铃声推送方面，支持 **Wi-Fi 和数据线**两种方式，传完直接出现在 iPhone 的"铃声"设置里，不需要任何中间步骤。

## 主要功能一览

- **自由剪辑**：波形可视化，拖拽精确选取片段，不猜不碰运气
- **音效与语音叠加**：可以加趣味音效或录制个人语音，铃声更有辨识度
- **淡入淡出**：自动让铃声开头结尾平滑过渡，不突兀
- **多格式导出**：可保存为 M4R（iPhone 标准铃声格式）或 MP3，留存备用
- **1 Click 推送**：Wi-Fi 连接即可传输，数据线也支持，不依赖 iTunes
